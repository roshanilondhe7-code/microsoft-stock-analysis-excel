# Microsoft Stock Data Visualization (Excel)

This project analyzes Microsoft stock data from December 2020 to November 2021 using Microsoft Excel.

## What I did
- Analyzed daily stock price movements
- Compared daily price increases vs decreases
- Visualized average monthly closing prices
- Examined trading volume distribution and trends
- Explored relationships between price volatility and trading volume
- Created a final one-page report using multiple charts

## Tools Used
- Microsoft Excel
- Data visualization techniques

## Purpose
This project was completed as a learning exercise to better understand how data visualizations are used to explore trends, relationships, and patterns in real-world financial data.
